### 3. EXHAUSTIVE ANIMATION SYSTEM (Framer Motion)
Animations are the absolute core of this project. They must feel physical, fluid, and highly polished. Implement the following exactly:

**A. The "Reveal" Wrapper Component (Scroll Triggered Fades)**
Every major text block and grid item must be wrapped in a `<Reveal>` component that handles scroll-triggered entrance animations.
- Setup: Use `initial={{ opacity: 0, y: 20 }}` and `whileInView={{ opacity: 1, y: 0 }}`.
- Viewport: `viewport={{ once: true, margin: "-40px" }}`.
- Transition: `transition={{ duration: 0.9, delay: props.delay || 0, ease: [0.22, 1, 0.36, 1] }}`.
- Grid Items: When rendering maps/grids, pass a staggered delay: `<Reveal delay={index * 0.05}>`.

**B. Scroll Parallax (Hero Section)**
Use `useScroll` and `useTransform` mapped to a container ref for background and foreground parallax.
- `const ref = useRef(null);`
- `const { scrollYProgress } = useScroll({ target: ref, offset: ["start start", "end start"] });`
- Dashboard element moving UP: `const dashY = useTransform(scrollYProgress, [0, 1], [0, 160]);`
- Floating coin element moving UP (negative Y): `const coinY = useTransform(scrollYProgress, [0, 1], [0, -120]);`
- Apply to `motion.div` and `motion.img` via `style={{ y: dashY }}`.

**C. Hover Springs (Cards and Images)**
Use physics-based springs for interactions, avoiding linear or standard ease tweens.
- Standard Card Hover (Lift up): `whileHover={{ y: -6 }}` or `whileHover={{ y: -8 }}`.
- Standard Image Hover (Scale up): `whileHover={{ scale: 1.02 }}`.
- Slide Hover (List items): `whileHover={{ x: 6 }}`.
- **The Spring Physics**: Every hover must use `transition={{ type: "spring", stiffness: 200, damping: 20 }}` (for Y/X movement) or `stiffness: 150` for scaling.

**D. SVG Path Drawing (Sparkline Charts)**
For the market line charts, the path must draw itself when entering the viewport.
- `<motion.path d="..." fill="none" stroke="currentColor" strokeWidth="1.5" />`
- Animation props: `initial={{ pathLength: 0 }}`, `whileInView={{ pathLength: 1 }}`.
- Viewport: `viewport={{ once: true }}`.
- Transition: `transition={{ duration: 1.6, ease: "easeInOut" }}`.

**E. AnimatePresence & Height Animations (FAQ Accordion)**
- `<motion.div initial={false} animate={{ height: isOpen ? "auto" : 0, opacity: isOpen ? 1 : 0 }} transition={{ duration: 0.5, ease: [0.22, 1, 0.36, 1] }} className="overflow-hidden">`
